Quick Theory Recap

  • Web Development is about creating websites and web applications.
  • Front-end uses HTML, CSS and JavaScript to build what users see.
  • HTML gives structure, CSS gives style, JS adds behaviour.
  • Web pages are rendered by browsers like Chrome, Edge, Firefox.
